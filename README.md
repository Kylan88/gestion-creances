# Gestion de Créances Pro

Application professionnelle de recouvrement & gestion de clients  
Développée à Abidjan, Côte d'Ivoire

## Fonctionnalités
- Authentification complète + avatar persistant
- Dashboard avec statistiques
- Gestion clients (CRUD + recherche + filtres)
- Formulaire ultra-moderne (React Hook Form + Zod)
- Téléphone international avec drapeau
- Dark/Light mode fluide avec toggle animé
- 100% responsive

## Tech Stack
**Frontend** : React 18 + Vite + Tailwind + Framer Motion  
**Backend** : Flask + SQLite  
**Libs** : react-phone-number-input • zod • lucide-react
